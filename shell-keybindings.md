# Shell keybindings & commands (fzf / zoxide / Atuin)

- `CTRL-T` — fzf file/dir picker; inserts selection into the command line.
- `CTRL-R` — fzf history search (enhanced by Atuin’s full-text history).
- `z <keyword>` — jump to a directory learned by zoxide.
- `zi` — interactive directory jump (zoxide + fzf).
- `z -` — jump back to previous directory (zoxide).
- `UP` — Atuin-enhanced history navigation (context-aware).

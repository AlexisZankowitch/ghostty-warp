# Ghostty keybindings (Catppuccin Mocha + Fira Code)

- `Shift+Enter` → send literal newline (multiline commands)
- `Cmd+C` → copy to clipboard
- `Cmd+V` → paste from clipboard
- `Cmd+=` → increase font size by 1
- `Cmd+-` → decrease font size by 1
- `Cmd+0` → reset font size
- `Cmd+K` → clear screen
- `Cmd+T` → new tab
- `Cmd+W` → close surface
- `Cmd+D` → vertical split (right)
- `Cmd+Shift+D` → horizontal split (down)
- `Cmd+Shift+Enter` → toggle split zoom
- `Cmd+1` … `Cmd+9` → switch to tab 1–9
